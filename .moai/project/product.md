# Paste Trader - Product Vision

## 프로젝트 개요

**프로젝트명**: paste-trader
**유형**: 웹 브라우저 애플리케이션
**버전**: 0.1.0 (초기 개발)

### 미션

트레이더의 방법론을 AI 기반 자동화 워크플로우로 변환하여, 개인 투자자가 전문 트레이더의 전략을 손쉽게 모방하고 실행할 수 있도록 지원한다.

### 비전

YouTube 영상, PDF 자료 등 다양한 형태의 트레이딩 방법론 콘텐츠를 붙여넣기만 하면, LLM이 자동으로 분석하고 DAG 패턴의 실행 가능한 파이프라인으로 변환하는 혁신적인 트레이딩 자동화 플랫폼.

---

## 핵심 문제 정의

### 현재 상황 (AS-IS)

1. **정보 과부하**: 수많은 트레이딩 방법론 콘텐츠가 존재하지만, 실제 적용하기 어려움
2. **수동 프로세스**: 트레이더의 전략을 따라하려면 수작업으로 데이터 수집, 분석, 판단 필요
3. **시간 제약**: 오전 9:30, 오후 15:00 등 정해진 시간에 분석해야 하는 부담
4. **기술 장벽**: 자동화 시스템 구축에는 높은 기술적 진입 장벽 존재
5. **일관성 부족**: 감정적 판단으로 인한 전략 일관성 유지 어려움

### 목표 상태 (TO-BE)

1. **원클릭 자동화**: 방법론 콘텐츠 붙여넣기만으로 워크플로우 자동 생성
2. **지능형 파이프라인**: LLM이 방법론을 분석하고 DAG 패턴으로 실행 계획 수립
3. **스케줄 기반 실행**: 설정된 시간에 자동으로 데이터 수집 및 분석
4. **적합 종목 탐색**: 방법론 조건에 맞는 종목 자동 필터링
5. **판단 지원**: 매수/매도 결정을 위한 객관적 데이터 제공

## 핵심 기능

### 1. 콘텐츠 파싱 및 분석 (3가지 입력 소스)

웹 UI에서 트레이딩 전략을 붙여넣기(Paste)하는 3가지 방법을 제공합니다:

#### 1.1 단순 텍스트 입력
- **용도**: 직접 작성하거나 복사한 트레이딩 방법론 텍스트
- **처리**: LLM이 자연어 분석을 통해 전략 요소 추출
- **예시**: "RSI가 30 이하이면 매수, 70 이상이면 매도"

#### 1.2 PDF 파일 업로드
- **용도**: 트레이딩 교재, 보고서, 전자책 형태의 방법론
- **처리**: PDF 파싱 → 텍스트 추출 → LLM 분석
- **지원**: 텍스트 기반 PDF (이미지 PDF는 OCR 필요)

#### 1.3 YouTube 영상 URL
- **용도**: 트레이딩 강의, 전략 설명 영상
- **처리**: 자막 추출 → 텍스트 변환 → LLM 분석
- **지원**: 자동 생성/수동 자막 포함 영상

#### LLM 분석 결과
모든 입력 소스에서 다음 핵심 요소를 추출합니다:
- 진입 조건 (Entry Criteria)
- 청산 조건 (Exit Criteria)
- 기술적 지표 (Technical Indicators)
- 시간 조건 (Time Constraints)
- 리스크 관리 규칙 (Risk Management)

### 2. 워크플로우 자동 생성 및 시각화

#### 자동 생성
- **DAG 패턴**: 방향성 비순환 그래프로 실행 순서 정의
- **노드 타입**: 도구(Tool), 에이전트(Agent), 조건(Condition)
- **어댑터**: 노드 간 데이터 형식 변환
- **YAML 기반**: 읽기 쉽고 수정 가능한 워크플로우 정의

#### 워크플로우 시각화 (React Flow 기반)
메타 LLM이 생성한 워크플로우를 시각적으로 표현하고 편집할 수 있습니다:

- **인터랙티브 에디터**: 드래그 앤 드롭으로 노드 배치 및 연결
- **커스텀 노드**: 노드 타입별 맞춤형 UI 컴포넌트
  - Tool Node: 도구 실행 노드 (입력/출력 포트)
  - Agent Node: LLM 에이전트 노드 (프롬프트 미리보기)
  - Condition Node: 분기 조건 노드 (다중 출력 포트)
  - Adapter Node: 데이터 변환 노드
- **실시간 상태**: Zustand 기반 상태 관리로 즉각적인 UI 반영
- **연결선(Edge)**: 노드 간 데이터 흐름을 시각적으로 표현
- **미니맵**: 복잡한 워크플로우의 전체 구조 조망
- **줌/팬**: 자유로운 캔버스 탐색

### 3. 메타 LLM 엔진

- **ReAct 패턴**: 추론(Reasoning) + 행동(Action) 기반 에이전트
- **도구 활용**: grep, AST-Grep 기반 코드 분석
- **다중 에이전트**: 복잡한 작업의 분업 처리
- **동적 생성**: 필요에 따라 새로운 에이전트/도구 생성

### 4. 스케줄 기반 실행

- **정기 실행**: 오전(9:30), 오후(15:00) 자동 분석
- **이벤트 트리거**: 특정 조건 발생 시 알림
- **수동 실행**: 언제든 온디맨드 분석 가능

### 5. 종목 스크리닝

- **조건 기반 필터링**: 방법론에 정의된 조건으로 종목 탐색
- **실시간 데이터**: 최신 시세 및 지표 반영
- **순위화**: 조건 부합도에 따른 종목 순위

### 6. 매매 판단 지원

- **시그널 생성**: 매수/매도/관망 시그널
- **근거 제시**: 판단의 논리적 근거 설명
- **리스크 분석**: 예상 손익 및 리스크 평가

## 사용자 시나리오

### 시나리오 1: 신규 전략 등록

1. 사용자가 YouTube 트레이딩 강의 URL을 붙여넣음
2. 시스템이 영상 내용을 분석하여 트레이딩 방법론 추출
3. 추출된 방법론을 YAML 워크플로우로 변환
4. 사용자가 워크플로우 검토 및 수정
5. 스케줄 설정 후 활성화

### 시나리오 2: 정기 분석

1. 오전 9:30 스케줄 트리거 발동
2. 등록된 워크플로우 순차 실행
3. 각 방법론 조건에 맞는 종목 스크리닝
4. 분석 결과 및 추천 종목 리스트 생성
5. 사용자에게 알림 발송

### 시나리오 3: 실시간 모니터링

1. 사용자가 관심 종목 등록
2. 등록된 전략 조건 실시간 모니터링
3. 진입/청산 조건 충족 시 즉시 알림
4. 상세 분석 리포트 제공

## 경쟁 우위

### 차별화 요소

1. **원스톱 자동화**: 콘텐츠 입력부터 실행까지 통합 플랫폼
2. **LLM 기반 지능형 분석**: 자연어 방법론을 실행 가능한 코드로 변환
3. **유연한 커스터마이징**: YAML 기반으로 누구나 워크플로우 수정 가능
4. **Multi-Provider LLM**: 다양한 LLM 제공자 지원으로 비용 최적화
5. **오픈 도구 생태계**: 커뮤니티 기반 도구/에이전트 공유

### 기존 솔루션 대비 장점

| 기능 | 기존 솔루션 | paste-trader |
|------|-------------|--------------|
| 전략 입력 | 코드 직접 작성 | 자연어 붙여넣기 |
| 워크플로우 생성 | 수동 설정 | AI 자동 생성 |
| LLM 통합 | 제한적/없음 | 핵심 기능 |
| 커스터마이징 | 복잡한 설정 | 직관적 YAML |

## 프로젝트 범위

### 1단계 (MVP)

- 텍스트 기반 방법론 입력 및 파싱
- 기본 워크플로우 생성 (5개 노드 이내)
- 단일 LLM Provider 지원
- 수동 실행 기능

### 2단계 (Beta)

- YouTube URL 파싱 지원
- 복잡한 워크플로우 (10개 노드 이상)
- Multi-Provider LLM 지원
- 스케줄 기반 실행
- 실시간 알림

### 3단계 (Production)

- PDF 파싱 지원
- 워크플로우 마켓플레이스

- 브로커 API 연동
- 커뮤니티 기능
